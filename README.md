# Tiger compiler

## Introduction
This is a C++ implementation of Tiger Compiler, described in *Modern Compiler Implementation in C*.


## How to make
1. The very first make
    ```bash
    make objs # To generate bison and flex files
    make tiger
    ```
2. After first make
    ```bash
    make tiger
    ```
